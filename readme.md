# Team Finder

Платформа для поиска команды и проектов. Бэкенд-приложение на Django. Вариант 2 – навыки пользователей, фильтрация по навыкам.

## Стек технологий

- **Backend:** Python 3.11, Django 5.2
- **База данных:** PostgreSQL 13
- **Контейнеризация:** Docker, Docker Compose
- **Статика:** HTML, CSS (предоставлены в репозитории)

## Требования

- Docker и Docker Compose (для быстрого разворачивания)
- Либо локально: Python 3.11, PostgreSQL
